# Universal Quantum Information Framework (UQIF) Repository

This repository contains the publication materials for the Universal Quantum Information Framework (UQIF), a theoretical framework exploring the connections between quantum information, consciousness, and fundamental physics.

## Contents

- `uqif.pdf` - The fully typeset LaTeX PDF of the UQIF paper with proper equation rendering and all figures included
- `uqif_v23_*.png` - Simulation figures referenced in the paper:
  - `uqif_v23_chsh_nonlocality_pennylane.png` - CHSH nonlocality simulation using PennyLane
  - `uqif_v23_coherence_decay_qutip.png` - Coherence decay simulation using QuTiP
  - `uqif_v23_penrose_or_threshold_mc.png` - Penrose-Orch OR threshold Monte Carlo simulation
  - `uqif_v23_sensitivity_heatmap.png` - Sensitivity analysis heatmap
- `uqif_v23_enhanced_simulations.py` - Python script for running the enhanced simulations
- `uqif_v23_*.csv` - Data files generated by the simulations:
  - `uqif_v23_chsh_nonlocality_data.csv`
  - `uqif_v23_or_monte_carlo_data.csv`
  - `uqif_v23_qutip_coherence_data.csv`
  - `uqif_v23_sensitivity_data.csv`

## Description

The Universal Quantum Information Framework proposes a novel approach to understanding quantum mechanics through the lens of information theory, with particular attention to the role of consciousness in quantum measurement processes. This framework integrates concepts from:

- Quantum information theory
- Many-Worlds Interpretation
- Orchestrated Objective Reduction (Orch-OR)
- Non-locality and Bell inequalities
- Quantum coherence and decoherence

The paper presents mathematical formulations, simulation results, and theoretical implications for fields ranging from quantum computing to consciousness studies.

## Usage

The PDF file `uqif.pdf` contains the complete paper with all equations properly typeset and figures embedded. The accompanying data files and simulation script allow for reproduction and extension of the numerical results presented in the paper.

To run the simulations:
```bash
python3 uqif_v23_enhanced_simulations.py
```
